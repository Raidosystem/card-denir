# Card de Presença Digital

Um cartão de presença digital estilizado para link de bio do Instagram, com botões para WhatsApp, Instagram, Facebook e Email.

## Tecnologias

- React
- TypeScript
- Framer Motion
- Vite

## Visual

Este cartão de presença apresenta um design moderno e tecnológico com:

- Tema nas cores laranja e preto
- Animações suaves
- Efeitos visuais ao passar o mouse
- Design responsivo

## Como personalizar

Para personalizar este cartão com suas informações:

1. Edite o arquivo `src/App.tsx` e atualize o objeto `userInfo`
2. Substitua a imagem em `src/assets/logo.jpg` pela sua
3. Atualize os links de contato com suas informações

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Gerar build de produção
npm run build

# Publicar no GitHub Pages
npm run deploy
```
